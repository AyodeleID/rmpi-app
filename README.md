# Real-Time Migration Pressure Index (RMPI)

## Description
The RMPI app is a Streamlit dashboard that visualises migration pressures across regions in real time. It aggregates economic, environmental and social indicators to provide a single score indicating migration pressure.

## Features
- **Interactive map** displaying RMPI scores by country or region.
- **Line charts** showing how the index evolves over time.
- **Filters** for selecting regions, time periods and indicator weights.
- **Data downloads** for further analysis.

## Running the App
1. Install the requirements with `pip install -r requirements.txt`.
2. Run `streamlit run app.py`.
3. Open the provided local URL to access the dashboard.

To try the live version, click the badge in the repository header.
